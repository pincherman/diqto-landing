Bien démarrer

Votre premier quart d'heure avec Diqto.

Ne préparez pas une démonstration. Prenez simplement une tâche que vous devez vraiment terminer aujourd'hui.

1

Choisissez quelque chose de réel

Un devis à envoyer, une facture à préparer, une note de séance ou un client dont vous devez reprendre le dossier. Plus le cas est concret, plus l'essai sera utile.

2

Parlez comme à un collègue

Donnez le nom du client, ce que vous avez fait, les prestations ou travaux concernés, les montants déjà connus et ce qui reste à vérifier.

Exemple« Chez Martin, remplacement du chauffe-eau 200 litres, dépose de l'ancien, pose et raccordement. Je dois encore vérifier le prix du matériel. »
3

Regardez ce que Diqto a compris

Vérifiez le client, le type de document, les lignes proposées et les informations manquantes. Corrigez directement ce qui doit l'être.

4

Relisez avant de partager

Le document reste un brouillon jusqu'à votre validation. Contrôlez les montants, les coordonnées et les libellés. Partagez seulement quand vous êtes satisfait.

Trois repères pour savoir si Diqto vous aide vraiment.

Vous avez moins retapé

Les informations déjà dites ou déjà connues n'ont pas dû être saisies une seconde fois.

Vous savez quoi faire ensuite

Le brouillon, l'information manquante ou la prochaine action sont clairement visibles.

Vous êtes resté maître du résultat

Rien n'a été partagé ou décidé sans que vous puissiez le relire et le modifier.
